# 🚀 Node.js Backend Learning Path (Beginner → Advanced)

This repository is a **complete hands-on journey** to becoming a **production-ready Node.js backend engineer**.

It is structured in **phases**, each building real-world backend skills.

---

## 📌 Learning Philosophy

* Learn by **building real systems**
* Focus on **production-level practices**
* Understand **how things work internally** (not just frameworks)
* Think like a **senior backend engineer**

---

## 🧱 Phase 1 — Core Node.js (No Framework)

📁 `phase-1-core-node/`

### 🎯 Goal

Understand how Node.js works under the hood.

### 📚 Topics

* HTTP module
* Request/Response lifecycle
* Manual routing
* JSON handling
* Streams (basic)

### 🛠 Project

* Build API without frameworks
